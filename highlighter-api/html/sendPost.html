This API is open source! Source code can be found at https://github.com/tonylizj/highlighter

To use this API, send a POST request to this address with three parameters: "text" containing a code block, "lang" containing one of

['typescript', 'c', 'cpp', 'csharp', 'python', 'java', 'go', 'julia', 'kotlin', 'haskell', 'lisp', 'lua', 'makefile', 'markdown', 'matlab', 'mongodb', 'objectivec', 'pascal', 'perl', 'php', 'r', 'racket', 'ruby', 'rust', 'scala', 'scheme', 'swift', 'visual-basic', 'json', 'latex', 'graphql', 'docker', 'markup', 'css', 'clike', 'javascript']

and "quality" containing one of

['medium', 'high', 'extreme']

Character limits: medium: 4000, high: 2000, extreme: 800
